<h1>
    This repository's purpose is to learn how to setup a new git repo
</h1>

<ol>
    <li>
        git init
    </li>
    <li>
        git add file.txt
    </li>
    <li>
        git commit -m "Message"
    </li>
    <li>
        git remote add origin url-to-github
    </li>
    <li>
        git push -u origin master
    </li>
</ol>

<h2>For pull request</h2>

<ol>
    <li>
        Fork repository
    </li>
    <li>
        git clone forked-repository-url
    </li>
    <li>
        Do changes
    </li>
    <li>
        git add .
    </li>
    <li>
        git commit -m "Message"
    </li>
    <li>
        git push
    </li>
    <li>
        Pull request
    </li>
</ol>
